# مستندات ویوئِرو

بیش از هر چیزی، از سفارش این قالب و مشتری عزیز ما بودن از شما سپاسگذاریم. شما می‌توانید مادام العمر آپدیت‌های این محصول را بصورت رایگان دریاف کنید. و برای پشتیبانی تیم ما همواره پاسخگوی تماس‌های شما خواهد بود.

این راهنمایی بمنظور کمک به شما در هر مرحله از راه‌اندازی و سفارشی‌سازی قالب نوشته شده است. لطفاً مستندات را با دقت مرور کنید تا با نحوه‌ی ساخت این قالب آشنا شده و ویرایش و تغییر قالب را به درستی انجام دهید. برای سفارشی‌سازی این ‌قالب آشنایی با HTML، CSS و Vue JS مورد نیاز است.

شما در حال مطالعه مستندات مربوط به ویوئِرو 2.1.0 هستید. این محصول از تکنولوژی‌های زیر استفاده می‌کند:

  • vue (v3.2+) API ترکیبی
  • Typescript (v4+) مناسب برای برنامه های کاربردی جاوا اسکریپتی در مقیاس بزرگ برای هر مرورگری
  • vitejs (v2.6+) ابزار ساخت و توسعه
  • nodejs (v14+) پلتفذم توسعه
  • آخرین نسخه‌ی bulma (v0.9.3+) ادغام با sass
  • نمونه پاسخ‌های HTTP Rest API بمنظور کمک به شما در ساخت API خودتان
  • ایمیج‌های آماده به استفاده‌ی داکر برپایه‌ی bitnami
  • پشتیبانی از yarn، npm (v7+) و pnpm
  • eslint، stylelint و prettier از پیش تنظیم شده‌اند

# پشتیبانی

اگر هنگام ویرایش این قالب با مشکلی مواجه گشتید یا می‌خواهید در مورد قالب سوال بپرسید، با ما با این آدرس support@vuero.ir تماس بگیرید.

# پیش نیازها

  1. یک ویرایشگر خوب برای کدنویسی
    VSCode دارای تنظیمات از پیش پیکربندی شده است
  2. مرورگر مناسب (کروم، اج یا فایرفاکس و غیره)
  3. نود جی اس نسخه‌ی LTS بالای ۱۴ همراه با npm بالای ۷
  4. آشنایی با Typescript (نسخه‌ی بالاتر از ۴) (نباید بصورت فراگیر نصب شود)

# نصب nodejs

ابتدا بررسی کنید که آیا قبلاً node و npm را نصب کرده اید یا خیر. برای بررسی اینکه آیا Node.js را نصب کرده اید ، این دستور را در ترمینال خود اجرا کنید:

node -v

برای تأیید اینکه npm را نصب کرده اید می توانید این دستور را در ترمینال خود اجرا کنید:

npm -v

اگر Node روی دستگاه شما نصب نشده است ، می توانید به وب سایت رسمی nodejs.org بروید ، و بسته به سیستم عامل خود نسخه‌ای را برای نصب انتخاب کنید:

  • node.js و npm را در Windows ، Linux یا Mac OSX نصب کنید

# نسخه npm را بررسی کنید

npm یک پروژه جداگانه از Node.js است و بروزرسانی‌های سریع‌تر دارد. در نتیجه ، حتی اگر Node.js (و به تبع اون npm) را نصب کرده باشید، احتمالاً باید npm خود را به روز کنید. خوشبختانه ، npm می داند چگونه خود را به روز کند! برای به روز رسانی npm خود ، این را در ترمینال خود تایپ کنید:

npm install --global npm@latest

# نصب Yarn

ما شخصاً Yarn را توصیه می کنیم زیرا فکر می کنیم بهتر از npm است. اگر می خواهید از Yarn استفاده کنید ، به خاطر داشته باشید که نباید ایندو را باهم مخلوط کنید. یا از Yarn یا Npm در یک پروژه استفاده می کنید، هر دو را باهم هرگز.

npm install --global yarn

# نصب متعلقات

برای راه اندازی قالب و شروع نصب وابستگی های پروژه ، یکی از دستورات زیر را اجرا کنید:

# using npm >7
npm install --legacy-peer-deps

# using pnpm
pnpm install

# using yarn
yarn install

# اجرای سرور توسعه

برای راه اندازی سرور توسعه ، یکی از دستورات زیر را اجرا کنید:

# using npm >7
npm run dev

# using pnpm
pnpm dev

# using yarn
yarn dev

هشدار

در هنگام نصب در صورت برخورد با هر مشکلی ابتد صفحه‌ی مشکلات رایج را بررسی کنید سپس در صورت لزوم با ما تماس بگیرید

نکته

در مرورگر خود آدرس http://localhost:3000/ را باز کنید تا پیش نمایش ویوئِرو را ببینید.

اگر از مرورگر Chrome استفاده می کنید ، توصیه می کنیم افزونه vuejs-devtools را نصب کنید

می توانید تاریخچه تغییرات نسخه را در اینجا و یا داخل پوشه دانلود شده ببینید یا می‌توانید تغییرات را در صفحه فروش تم بررسی کنید.

مجدد از خرید این تم بسیار متشکرم. همانطور که در ابتدا گفتم، اگر سوالی در مورد این قالب دارید، خوشحال می‌شویم به شما کمک کنیم. نمی‌توانیم قول صد در صد بدهیم، اما ما تمام تلاش خود را برای کمک و پشتیبانی از شما انجام می‌دهیم. اگر هم سوال کلی تری در رابطه با این قالب یا درخواست اجرای پروژه و همکاری دارید، می‌توانید سوال خود را در یک ایمیل به ما مطرح کنید.
